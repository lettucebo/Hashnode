#### Firefox Setting for myself

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1776070328976/13aaf799-258b-4c9e-a45b-326ea3c629fd.png)

Firefox

我是忠實的火狐愛用者，其中有一個很大的原因是因為: Tab Mix Plus 這個擴充套件，實在提供了很多功能；嘗試換過 Chrome、Edge 但都唯獨缺少此套件所提供的功能，也無替代方案，所以全數放棄

但是在 Firefox 新版上 (Firefox Quantum) 就無法使用也不在提供更新了，所以實在沒辦法只好找了一下一些相關替代設定，以下紀錄相關資訊，以供日後使用

其中要直接修改 Firefox 的設定，開啟 **about:config**，並於搜尋欄中搜尋以下設定關鍵字

#### 頁籤設定

1. 書籤列點擊我的最愛時開啟於新頁籤：

```
browser.tabs.loadBookmarksInTabs: true
```

2. 搜尋欄搜尋時開啟於新頁籤：

```
browser.search.openintab: true
```

3. 連結位於不同網域時開啟於新頁籤：

```
browser.link.open_newwindow: 3
```

4. 於網址列輸入後開啟於新頁籤：

```
browser.urlbar.openintab: true
```

這樣就可以在網址列、搜尋列或是開啟我的最愛時都是開啟於新頁籤，而不會是直接用原來的頁籤開啟，保留原來所瀏覽的資訊
